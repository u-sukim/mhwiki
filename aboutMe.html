1. 개요

안녕하세요! 제주대학교 컴퓨터교육과 24학번 김민혁입니다. 저는 "개발 언어는 단순한 도구일 뿐, 중요한 건 본질적인 알고리즘과 문제 해결 능력"이라는 철학을 가지고 개발에 임하고 있습니다. 특정 언어에만 치우치지 않고 다양한 언어와 기술을 접하며, 필요에 따라 유연하게 개발 스타일을 바꾸는 잡식성 개발자입니다. 특히 교육자로서 학생들에게 더 나은 코딩 교육 경험을 제공하기 위해 디자인적인 감각과 실용적인 개발 능력을 동시에 키우려고 노력 중입니다.

현재 개발자로서의 역량을 키우는 것만큼이나 중요한 또 하나의 과제가 있습니다. 바로 발표와 커뮤니케이션 능력을 기르는 것입니다. 사범대생으로서, 발표 울렁증을 극복하고 자신감 있는 발표를 하기 위해 지속적으로 발표 과제를 통해 성장하고 있습니다. "코딩 능력"과 "교육자로서의 자질" 두 가지를 모두 갖춘 개발자가 되는 것이 저의 목표입니다.

2. 소속

제주대학교 컴퓨터교육과에 소속되어 있습니다. 이곳에서 교육자로서의 기본기를 배우며, 컴퓨터 과학의 원리를 깊이 있게 탐구하고 있습니다. 학과에서는 코딩 교육법과 다양한 소프트웨어 개발 방법론을 배우며, 이를 실습과 프로젝트를 통해 몸에 익히고 있습니다.

또한, 제주대학교의 자유로운 학습 분위기 속에서 동료들과 함께 협력하고, 다양한 프로젝트를 통해 실전 감각을 키우고 있습니다. 팀 프로젝트와 발표를 통해 의사소통 능력과 협업 능력을 꾸준히 향상시키는 중입니다.

3. 스택

저는 아직 공부 중인 학생이지만, 다양한 기술 스택을 탐구하며 실력을 키우고 있습니다. 현재 다음과 같은 기술들을 학습하고 활용하고 있습니다:

메인 개발 언어: Python, C
공부 중인 언어: Java, Go, Javascript

프론트엔드: HTML, CSS, Javascript
- 웹 디자인 및 인터랙티브 UI 구현에 관심이 많습니다.

백엔드: Flask
- 가벼운 웹 서버 개발 및 API 설계 경험이 있습니다.

GUI 개발: tkinter, PyQT
- 데스크톱 애플리케이션 UI를 설계하고 구현하는 데 도전하고 있습니다.

게임 개발: Godot Engine, pygame
- 간단한 2D 게임 개발부터 시작해 창의적인 게임 개발을 목표로 하고 있습니다.

저는 위의 기술 스택을 기반으로 다양한 프로젝트를 시도하며, 실무 감각과 문제 해결 능력을 키우고자 합니다.

4. 포트폴리오

현재는 포트폴리오를 구성하고 있는 단계입니다. 하지만 아래와 같은 경험을 통해 포트폴리오를 채우고 있습니다:

1. 교육 봉사 경험:
- 중학생을 대상으로 하는 수업: 중학생을 대상으로 하기 때문에 흥미를 잃지 않게 프로젝트 중심 수업을 설계함과 동시에 문법 내용도 개발에 무조건 필요한 부분은 놓치지 않고 모두 다루려고 노력하였습니다.
- 학생들의 눈높이에 맞춘 다양한 프로젝트:학생들이 중간에 흥미를 잃지 않을 정도의 너무 쉽지도 어렵지도 않은 문제를 직접 제작하여 학생들이 일상에서 마주치는 문제를 해결하는 방법으로서 프로그래밍을 받아들이게끔 수업을 설계했습니다.

2. 개발 경험:
- 학생들의 학습에 대한 흥미 유발을 목적으로 한 게임 개발 경험: 렌파이와 requests 모듈을 사용해 openai의 인공지능 모델을 베이스로 한 대화형 주관식 문제 채점 시스템을 개발해 사범대 교육페스티벌에 참가한 경험이 있습니다.

미래 계획:
저는 다양한 프로젝트를 통해 실력을 검증하고, 이를 체계적으로 정리하여 포트폴리오를 완성할 계획입니다. 또한, 게임 개발과 교육용 소프트웨어 제작에 집중하여 창의적이고 실용적인 결과물을 만들고 싶습니다. 물론 다른 프로그램 개발에도 관심이 많지만, 학생들을 가르치기 위한 학과에 다니는 만큼 PBL을 만들고 싶은 욕심이 있습니다.
